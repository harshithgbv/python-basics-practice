## How to Run
Download or Clone this Repository
ie.  git clone https://github.com/harshithgbv/python-basics-practice.git
in terminal/command prompt(cmd):
Navigate to the project folder:
cd python-basics-practice
TO Run Calculator program:
type in cmd:
python calculator.py

TO Run Even/Odd Checker:
type in cmd:
python even_odd.py

TO Run Simple Interest Calculator:
type in cmd:
python simple_interest.py
then :
Enter the required inputs when prompted and 
View results in the terminal.
